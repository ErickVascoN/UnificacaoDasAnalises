🚀 COMO USAR
═══════════════════════════════════════════════════════════════════════════

1. Ativar ambiente:
   .\venv\Scripts\Activate.ps1

2. Instalar dependências:
   pip install -r requirements.txt

3. Executar:
   streamlit run app.py

4. Abrir browser:
   http://localhost:8501

5. Testar setores:
   Home → Clique em Corte/Produção/Faturamento
